Commande Git :git branch 'nom'

Accueil Commandes À propos

Description

La commande git branch'nom' permet de Crée une nouvelle branche, mais ne bascule pas dessus automatiquement.

Exemple d'utilisation


            $ git branch'nom' https://github.com/utilisateur/projet.git

Cette commande sert à creer une nouvelle branche.
