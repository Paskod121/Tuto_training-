Commande Git :git checkout -b'nom'

Accueil Commandes À propos

Description

La commande git checkout -b'nom' permet de Crée une nouvelle branche et bascule immédiatement dessus.

Exemple d'utilisation


            $ git checkout -b'nom' https://github.com/utilisateur/projet.git

C'est une commande combinée qui remplace ces deux commandes :

  • git branch nom-de-la-branche # Création de la branche
  • git checkout nom-de-la-branche # Changement de branche
