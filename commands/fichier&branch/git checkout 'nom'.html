Commande Git :git checkout'nom'

Accueil Commandes À propos

Description

La commande git checkout'nom' permet de changer de branche et de basculer sur une autre branche existante.

Exemple d'utilisation


            $ git checkout'nom' https://github.com/utilisateur/projet.git

Cette commande sert à basculer sur une branche existante.
