Commande Git :git branch

Accueil Commandes À propos

Description

La commande git branch permet de gérer les branches dans Git

Exemple d'utilisation


            $ git branch https://github.com/utilisateur/projet.git

Cette commande sert à créer, lister ou supprimer des branches dans un dépôt Git.
Une branche, c’est comme une version parallèle du projet. Elle permet de travailler sur de nouvelles fonctionnalités sans toucher à la version principale (main ou master). .
