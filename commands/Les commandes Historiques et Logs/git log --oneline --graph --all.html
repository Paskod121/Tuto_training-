Commande Git :git log --oneline --graph --all

Accueil Commandes À propos

Description

La commande git log --oneline --graph --all permet d'afficher l’historique Git de manière résumée et visuelle, en montrant :chaque commit sur une seule ligne (--oneline),la structure des branches et fusions sous forme de graphe (--graph), tous les commits de toutes les branches (--all).

Exemple d'utilisation


            $git log --oneline --graph --all https://github.com/utilisateur/projet.git

Cette commande sert à afficher l'historique de manière condensée et graphique.
