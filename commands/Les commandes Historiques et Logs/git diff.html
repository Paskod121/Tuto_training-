Commande Git :git diff

Accueil Commandes À propos

Description

La commande git diff permet de voir les différences entre :les fichiers modifiés et non encore ajoutés (git add non fait),les fichiers stagés (dans la zone de staging), ou entre deux commits.

Exemple d'utilisation


            $ git diff https://github.com/utilisateur/projet.git

Cette commande est très utile pour comparer, revoir ou vérifier les changements dans un projet. .
