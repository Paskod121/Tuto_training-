Commande Git :git log

Accueil Commandes À propos

Description

La commande git log permet d'fficher l’historique des commits d’un dépôt Git.Elle te permet de voir qui a fait quoi, quand, et avec quel message.

Exemple d'utilisation


            $ git log https://github.com/utilisateur/projet.git

Cette commande sert à afficher l'historique des commits .
