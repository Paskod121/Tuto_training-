Commande Git :git status

Accueil Commandes À propos

Description

La commande git status permet d'afficher l’état actuel du dépôt Git(fichier modifiés, non suivis,etc.)

Exemple d'utilisation


        $ git status https://github.com/utilisateur/projet.git

Cette commande est utilisée pour voir :

  • Quels fichiers ont été modifiés.
  • Quels fichiers sont en attente d'ajout.
  • Quels fichiers sont prêts à être commit .
  • Si la branche locale est à jour avec la branche distante.
