Commande Git : git clone

Accueil Commandes À propos

Description

La commande git clone copie un dépôt existant vers un nouveau répertoire local. Elle est utilisée pour travailler sur un projet déjà existant.

Exemple d'utilisation


$ git clone https://github.com/utilisateur/projet.git

Cette commande clone le dépôt distant dans un répertoire local nommé projet.
