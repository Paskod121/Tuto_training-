Commande Git :git config --global user.email

Accueil Commandes À propos

Description

La commande git config --global user.email permet de définir l'adresse e-mail associée aux commits Git pour l'utilisateur sur l’ensemble des dépôts.

Exemple d'utilisation


         $ git config --global user.email https://github.com/utilisateur/projet.git

Cette commande sert à définir l'adresse email globalement .
