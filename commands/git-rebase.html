Commande Git : git rebase

Accueil Commandes À propos

Description

La commande git rebase réapplique les commits d'une branche sur une autre pour maintenir un historique linéaire.

Exemple d'utilisation


$ git rebase main

Cette commande réapplique les commits de la branche courante sur la branche main.
