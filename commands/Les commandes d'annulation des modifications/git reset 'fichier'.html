Commande Git :reset

Accueil Commandes À propos

Description

La commande reset permet de revenir en arrière dans l’historique Git, en déplaçant le pointeur HEAD (c’est-à-dire l’état actuel de ton projet) vers un commit précédent.

Exemple d'utilisation


            $git reset https://github.com/utilisateur/projet.git

Cette commande sert à annuler l'ajout d'un fichier à la zone de staging.
