Commande Git :git revert 'commit'

Accueil Commandes À propos

Description

La commande git revert 'commit' permet d'annuler un commit spécifique en créant un nouveau commit inverse.

Exemple d'utilisation


            $git revert 'commit' https://github.com/utilisateur/projet.git

    Ce que fait git revert
:
  • Elle ne modifie pas l’historique Git.
  • Elle ajoute un nouveau commit qui annule les changements du commit ciblé.
