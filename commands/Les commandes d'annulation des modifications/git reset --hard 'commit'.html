Commande Git :git reset --hard'commit'

Accueil Commandes À propos

Description

La commande git reset --hard'commit' permet de revenir à un ancien état du projet, en effaçant complètement les modifications faites après un certain commit.

Exemple d'utilisation


            $git reset --hard'commit' https://github.com/utilisateur/projet.git

    Décomposition de la commande :
  • git reset : Repositionne l’historique (HEAD) sur un commit spécifique.
  • --hard : Supprime aussi les fichiers modifiés dans le dossier de travail et dans la zone de staging.
  • 'commit' : L’ID (ou hash) du commit vers lequel tu veux revenir
