Commande Git : git branch

Accueil Commandes À propos

Description

La commande git branch permet de lister, créer ou supprimer des branches dans un dépôt Git.

Exemple d'utilisation


$ git branch nouvelle-branche

Cette commande crée une nouvelle branche appelée nouvelle-branche.
