Commande Git : git merge

Accueil Commandes À propos

Description

La commande git merge fusionne une branche dans la branche courante.

Exemple d'utilisation


$ git merge branche-a-fusionner

Cette commande fusionne la branche branche-a-fusionner dans la branche courante.
