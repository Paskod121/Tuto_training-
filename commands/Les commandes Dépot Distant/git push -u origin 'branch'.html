Commande Git :git push -u origin 'branch'.

Accueil Commandes À propos

Description

La commande git push -u origin 'branch' permet d'envoyer la branche locale (et ses commits) vers le dépôt distant (souvent sur GitHub), pour la première fois.

Exemple d'utilisation


        $ git push -u origin 'branch'  https://github.com/utilisateur/projet.git

La décomposition de git push -u origin 'branch' :

  • git push → envoie tes commits vers le dépôt distant.
  • origin → nom du dépôt distant (par défaut, c’est souvent origin).
  • nom-de-branche → la branche que tu veux pousser (ex : main, develop, feature-login...).
