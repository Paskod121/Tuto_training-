Commande Git : git pull origin 'branch'.

Accueil Commandes À propos

Description

La commande git pull origin 'branch'permet de synchroniser un dépôt local avec un dépôt distant, en intégrant automatiquement les modifications distantes dans la branche locale.

Exemple d'utilisation


        $ g git pull origin 'branch' https://github.com/utilisateur/projet.git

Cette commande sert à récuperer les modifications du dépôt distant et les fusionne dans la branche actuelle
