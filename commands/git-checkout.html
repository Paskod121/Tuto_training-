Commande Git : git checkout

Accueil Commandes À propos

Description

La commande git checkout permet de basculer entre les branches ou de restaurer des fichiers dans l'état d'une révision spécifique.

Exemple d'utilisation


$ git checkout nouvelle-branche

Cette commande bascule vers la branche nouvelle-branche.
