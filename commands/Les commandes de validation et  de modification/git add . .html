Commande Git :git add.

Accueil Commandes À propos

Description

La commande git add . permet d'ajoute tous les fichiers modifiés, créés ou supprimés dans le répertoire courant à l’index .

Exemple d'utilisation


             $ git add. ttps://github.com/utilisateur/projet.git

Le . signifie "tout le répertoire courant".
Pour ajouter un fichier spécifique seulement on fait :"git add mon_fichier.txt"
