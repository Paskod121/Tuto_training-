Commande Git : git commit -m "Message"

Accueil Commandes À propos

Description

La commande git commit -m "Message" permet de sauvegarder les modifications dans l'historique du projet.

Exemple d'utilisation


             $ git commit -m "Message" ttps://github.com/utilisateur/projet.git

Le -m sert à ajouter un message court et clair qui explique le but du commit (ajout d’une fonctionnalité, correction d’un bug, etc.)
