Commande Git :git clone

Accueil Commandes À propos

Description

La commande git clone permet de copier un dépôt Git distant (comme sur GitHub, GitLab, etc.) vers un ordinateur local.

Exemple d'utilisation


            $ git clone https://github.com/utilisateur/projet.git

Cette commande sert à cloner un dépôt distant vers la machine locale .
