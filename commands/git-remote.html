Commande Git : git remote

Accueil Commandes À propos

Description

La commande git remote permet de gérer les dépôts distants associés à votre dépôt local.

Exemple d'utilisation


$ git remote add origin https://github.com/utilisateur/projet.git

Cette commande ajoute un dépôt distant nommé origin.
