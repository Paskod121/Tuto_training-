Commande Git : git mv

Accueil Commandes À propos

Description

La commande git mv renomme ou déplace des fichiers dans le dépôt.

Exemple d'utilisation


$ git mv ancien-nom.txt nouveau-nom.txt

Cette commande renomme le fichier ancien-nom.txt en nouveau-nom.txt.
