Commande Git : git rm

Accueil Commandes À propos

Description

La commande git rm supprime des fichiers du dépôt et de la zone de staging.

Exemple d'utilisation


$ git rm fichier.txt

Cette commande supprime le fichier fichier.txt du dépôt.
