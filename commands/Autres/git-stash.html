Commande Git : git stash

Accueil Commandes À propos

Description

La commande git stash met de côté les modifications en cours pour les appliquer plus tard.

Exemple d'utilisation


$ git stash

Cette commande sauvegarde les modifications en cours dans une pile temporaire.
