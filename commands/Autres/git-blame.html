Commande Git : git blame

Accueil Commandes À propos

Description

La commande git blame montre l'historique des modifications ligne par ligne pour un fichier.

Exemple d'utilisation


$ git blame fichier.txt

Cette commande affiche l'auteur et la date de chaque ligne du fichier fichier.txt.
