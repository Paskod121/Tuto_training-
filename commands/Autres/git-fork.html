Commande Git : git fork

Accueil Commandes À propos

Description

La commande git fork n'est pas une commande Git native, mais elle est utilisée pour désigner l'action de copier un dépôt distant (souvent via des plateformes comme GitHub) dans votre propre espace utilisateur.

Exemple d'utilisation

Pour forker un dépôt sur GitHub :

  1. Accédez au dépôt sur GitHub.
  2. Cliquez sur le bouton Fork en haut à droite.
  3. Le dépôt sera copié dans votre espace utilisateur.
