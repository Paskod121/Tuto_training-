Commande Git : git tag

Accueil Commandes À propos

Description

La commande git tag ajoute une étiquette (tag) à un commit spécifique.

Exemple d'utilisation


$ git tag -a v1.0 -m "Version 1.0"

Cette commande ajoute un tag annoté nommé v1.0 avec un message "Version 1.0".
