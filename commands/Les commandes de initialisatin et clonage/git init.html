Commande Git :git init

Accueil Commandes À propos

Description

La commande git init permet d'initialiser un dépôt Git dans un dossier. C’est la première commande à exécuter lorsqu’on veut utiliser Git dans un projet.

Exemple d'utilisation


$ git init https://github.com/utilisateur/projet.git

Cette commande sert à initialiser un dépôt Git dans le répertoire actuel .
