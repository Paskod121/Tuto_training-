Commande Git : git init

Accueil Commandes À propos

Description

La commande git init initialise un nouveau dépôt Git dans le répertoire courant. Elle crée une structure pour suivre les modifications de vos fichiers.

Exemple d'utilisation


$ mkdir mon-projet
$ cd mon-projet
$ git init

Cette commande crée un dépôt Git dans le dossier mon-projet.
