Commande Git :git config --global user.name

Accueil Commandes À propos

Description

La commande git config --global user.name permet de définir ou modifier le nom d'utilisateur utilisé par Git pour identifier l'auteur des commits. .

Exemple d'utilisation


         $ git config --global user.name https://github.com/utilisateur/projet.git

Cette commande sert à définir le nom d'auteur pour tous les commits. .
