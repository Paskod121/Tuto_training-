Commande Git : git fetch

Accueil Commandes À propos

Description

La commande git fetch récupère les modifications depuis un dépôt distant sans les intégrer dans la branche courante.

Exemple d'utilisation


$ git fetch origin

Cette commande récupère les modifications du dépôt distant origin.
