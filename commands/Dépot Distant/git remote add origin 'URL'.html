Commande Git :git remote add origin URL.

Accueil Commandes À propos

Description

La commande git remote add origin URL permet de lier ton dépôt local à un dépôt distant (GitHub, GitLab, Bitbucket...).

Exemple d'utilisation


        $ git remote add origin URL https://github.com/utilisateur/projet.git

Les Syntaxes de Cette commande sont :

  • origin : Nom donné au dépôt distant (par défaut, c'est toujours origin)
  • URL : L'adresse du dépôt distant (ex: GitHub).
