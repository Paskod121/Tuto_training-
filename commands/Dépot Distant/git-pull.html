Commande Git : git pull

Accueil Commandes À propos

Description

La commande git pull récupère et intègre les modifications depuis un dépôt distant dans la branche courante.

Exemple d'utilisation


$ git pull origin main

Cette commande récupère les modifications de la branche main du dépôt distant origin.
